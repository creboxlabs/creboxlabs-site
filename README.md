# creboxlabs-site

Marketing site for Crebox Labs LLC. Static, server-rendered Next.js + Tailwind CSS deployed on Vercel.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
